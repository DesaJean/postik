# Code of Conduct

This project and its community adopt the [Contributor Covenant, version 2.1][cc].
By participating you are expected to uphold it.

The full text — including standards of expected and unacceptable behavior, the
enforcement guidelines, and the four-tier consequence ladder — lives at the link
below. We follow it as published, with no project-specific modifications.

[Read the Contributor Covenant 2.1 →][cc]

## Reporting

Instances of conduct that violate the Covenant can be reported privately to the
project maintainers. Open a confidential GitHub issue, or email the address
listed in the project's GitHub profile. Reports are reviewed promptly and
investigated fairly; the reporter's identity is kept confidential.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
